## Следующие шаги для выкладки на GitHub:

### 1. Создайте репозиторий на GitHub

1. Зайдите на https://github.com
2. Нажмите кнопку **"+"** в правом верхнем углу
3. Выберите **"New repository"**
4. Заполните:
   - **Repository name**: `zamok-wordpress-theme` (или другое название)
   - **Description**: `WordPress тема для сайта вскрытия замков`
   - Выберите **Public** или **Private**
   - **НЕ** ставьте галочку "Initialize this repository with a README"
5. Нажмите **"Create repository"**

### 2. Подключите удаленный репозиторий

После создания репозитория GitHub покажет инструкции. Выполните в терминале:

```bash
git remote add origin https://github.com/ВАШ_USERNAME/zamok-wordpress-theme.git
```

Замените:
- `ВАШ_USERNAME` - на ваш GitHub username
- `zamok-wordpress-theme` - на название вашего репозитория

### 3. Отправьте код на GitHub

```bash
git push -u origin main
```

Если потребуется авторизация:
- GitHub может попросить ввести логин и пароль
- Или использовать Personal Access Token (рекомендуется)

### 4. Проверка

После успешной отправки зайдите на страницу вашего репозитория на GitHub - там должны быть все файлы проекта.
