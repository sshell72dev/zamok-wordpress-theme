# WordPress тема Zamok01

Интеграция верстки сайта вскрытия замков в WordPress с использованием ACF (Advanced Custom Fields).

## Структура темы

### Основные файлы
- `style.css` - основной файл темы с метаданными
- `functions.php` - функции темы, регистрация типов записей, меню
- `header.php` - шапка сайта
- `footer.php` - подвал сайта
- `index.php` - основной шаблон

### Шаблоны страниц
- `front-page.php` - главная страница
- `page-service.php` - шаблон страницы услуги
- `page-contacts.php` - шаблон страницы контактов
- `archive-review.php` - архив отзывов
- `archive-article.php` - архив статей
- `single-article.php` - отдельная статья

### Типы записей
1. **Услуги (service)** - страницы услуг
2. **Отзывы (review)** - отзывы клиентов
3. **Статьи (article)** - статьи блога

### Таксономии
- **Категории статей (article_category)** - категории для статей

## Установка

1. Скопируйте папку `zamok` в директорию `wp-content/themes/`
2. Активируйте тему в админ-панели WordPress
3. Установите плагин **Advanced Custom Fields (ACF)**
4. Импортируйте ACF поля из папки `acf-json`

## Импорт ACF полей

### Способ 1: Автоматический (рекомендуется)
1. Убедитесь, что папка `acf-json` существует в корне темы
2. Перейдите в ACF → Field Groups
3. Поля автоматически загрузятся из JSON файлов

### Способ 2: Ручной импорт
1. Перейдите в ACF → Tools → Import Field Groups
2. Скопируйте содержимое каждого JSON файла из папки `acf-json`
3. Вставьте в поле импорта и нажмите "Import"

## Настройка меню

1. Перейдите в **Внешний вид → Меню**
2. Создайте меню и назначьте его в:
   - Главное меню
   - Услуги в футере
   - Информация в футере

## Создание страниц

### Главная страница
1. Создайте страницу "Главная"
2. Установите её как главную: **Настройки → Чтение → Главная страница**
3. Заполните ACF поля для главной страницы

### Страница услуги
1. Создайте страницу
2. Выберите шаблон "Страница услуги"
3. Заполните ACF поля

### Страница контактов
1. Создайте страницу "Контакты"
2. Выберите шаблон "Контакты"
3. Заполните ACF поля

## Создание записей

### Услуги
1. Перейдите в **Услуги → Добавить новую**
2. Заполните заголовок, описание, добавьте изображение
3. Заполните ACF поля

### Отзывы
1. Перейдите в **Отзывы → Добавить новый**
2. Заполните заголовок (имя клиента), текст отзыва
3. Заполните ACF поля (марка автомобиля, время вскрытия)

### Статьи
1. Перейдите в **Статьи → Добавить новую**
2. Заполните заголовок, содержание, добавьте изображение
3. Выберите категорию статьи
4. Заполните ACF поля

## Структура папок

```
zamok/
├── acf-json/          # JSON файлы ACF полей
├── css/              # Стили
├── js/               # Скрипты
├── img/              # Изображения
├── fonts/            # Шрифты
├── style.css         # Основной файл темы
├── functions.php     # Функции темы
├── header.php        # Шапка
├── footer.php        # Подвал
├── index.php         # Основной шаблон
├── front-page.php    # Главная страница
├── page-service.php  # Страница услуги
├── page-contacts.php # Страница контактов
├── archive-review.php # Архив отзывов
├── archive-article.php # Архив статей
└── single-article.php # Отдельная статья
```

## Примечания

- Все изображения должны быть в папке `img/`
- Пути к изображениям генерируются функцией `zamok01_get_image_url()`
- Для работы форм обратной связи необходимо добавить обработчики в `functions.php`
- Убедитесь, что все CSS и JS файлы находятся в соответствующих папках

## Поддержка

При возникновении проблем проверьте:
1. Установлен ли плагин ACF
2. Импортированы ли ACF поля
3. Правильно ли настроены меню
4. Существуют ли все необходимые файлы в папках css, js, img
