# Инструкция по импорту ACF полей

## Автоматический импорт (рекомендуется)

ACF автоматически загружает поля из папки `acf-json` при наличии плагина ACF.

### Шаги:
1. Убедитесь, что плагин **Advanced Custom Fields** установлен и активирован
2. Убедитесь, что папка `acf-json` существует в корне темы
3. Перейдите в **ACF → Field Groups**
4. Поля должны автоматически появиться в списке

Если поля не появились:
- Проверьте права доступа к папке `acf-json`
- Убедитесь, что JSON файлы имеют правильный формат
- Попробуйте перезагрузить страницу

## Ручной импорт через интерфейс ACF

1. Перейдите в **ACF → Tools → Import Field Groups**
2. Откройте файл из папки `acf-json` (например, `group_5f8a9b2c3d4e5.json`)
3. Скопируйте всё содержимое файла
4. Вставьте в поле "Import Field Groups"
5. Нажмите кнопку "Import"

Повторите для каждого JSON файла:
- `group_5f8a9b2c3d4e5.json` - Настройки сайта
- `group_main_page.json` - Главная страница
- `group_service_page.json` - Страница услуги
- `group_review.json` - Отзыв
- `group_service.json` - Услуга
- `group_article.json` - Статья
- `group_contacts_page.json` - Страница контактов

## Проверка импорта

После импорта проверьте:

1. **Настройки сайта**:
   - Перейдите в **Настройки сайта** (должна появиться в меню админ-панели)
   - Проверьте наличие всех полей

2. **Главная страница**:
   - Откройте главную страницу для редактирования
   - Внизу страницы должны появиться ACF поля

3. **Страница услуги**:
   - Создайте страницу и выберите шаблон "Страница услуги"
   - Проверьте наличие ACF полей

4. **Типы записей**:
   - Создайте новую запись типа "Отзыв", "Услуга" или "Статья"
   - Проверьте наличие соответствующих ACF полей

## Решение проблем

### Поля не отображаются
- Убедитесь, что плагин ACF активирован
- Проверьте, что JSON файлы находятся в правильной папке
- Очистите кеш WordPress (если используется плагин кеширования)

### Ошибки при импорте
- Проверьте формат JSON файлов (они должны быть валидными)
- Убедитесь, что кодировка файлов UTF-8
- Попробуйте импортировать файлы по одному

### Поля не сохраняются
- Проверьте права доступа к папке `acf-json` (должны быть права на запись)
- Убедитесь, что функция `zamok01_acf_json_save_point` работает корректно

## Структура ACF полей

### Настройки сайта (Options)
Все глобальные настройки сайта находятся в разделе "Настройки сайта" в админ-панели.

### Страницы
Каждая страница имеет свои ACF поля в зависимости от шаблона.

### Типы записей
- **Отзывы** - поля: марка автомобиля, время вскрытия, фотография
- **Услуги** - поля: описание, цена, meta теги
- **Статьи** - поля: описание, meta теги

## Дополнительная информация

После успешного импорта все поля будут доступны для редактирования в соответствующих разделах WordPress.

Для изменения полей:
1. Перейдите в **ACF → Field Groups**
2. Выберите нужную группу полей
3. Внесите изменения
4. Сохраните - изменения автоматически сохранятся в JSON файл
