## Первоначальная настройка

### 1. Активация темы
При активации темы автоматически создаются:
- Основные страницы (Главная, Контакты, страницы услуг)
- Категории статей
- Примеры услуг, отзывов и статей
- Базовые настройки ACF

**Если контент не создался автоматически:**
1. Перейдите в админ-панель WordPress
2. Откройте **Инструменты → Создать страницы Zamok01**
3. Нажмите на ссылку для создания страниц

### 2. Установка плагинов
Убедитесь, что установлен и активирован плагин **Advanced Custom Fields (ACF)**.

## Настройки сайта (ACF)

**Путь:** Админ-панель → Настройки сайта

### Основные настройки

#### Контактная информация
- **Основной телефон** — телефон, отображаемый на сайте (например: `+7 (495) 514-53-50`)
- **Основной email** — email для получения заявок (например: `zamok-01@yandex.ru`)
- **Телефоны в футере** — список дополнительных телефонов (можно добавить несколько)

#### Социальные сети
- **Ссылка VK** — ссылка на страницу ВКонтакте
- **Ссылка WhatsApp** — ссылка на WhatsApp (формат: `https://wa.me/74955145350`)
- **Ссылка Telegram** — ссылка на Telegram

#### Информация в футере
- **Информация в футере** — текст, отображаемый в нижней части сайта
- **Заголовок щита в футере** — заголовок блока с информацией об ассоциации
- **Изображение щита** — логотип/изображение ассоциации
- **Текст под щитом** — описание ассоциации

#### Ссылки
- **Ссылка на политику конфиденциальности** — URL страницы с политикой
- **Ссылка на карту сайта** — URL карты сайта
- **Ссылка на создателя** — ссылка на разработчика (опционально)

#### Справочники
- **Марки автомобилей** — список марок для формы отзывов
  - Значение (например: `mercedes`)
  - Название (например: `Mercedes`)
- **Время вскрытия** — варианты времени для формы отзывов
  - Значение (например: `2`)
  - Название (например: `2 минуты`)

#### Блок действия (по умолчанию)
- **Заголовок блока действия** — заголовок блока с призывом к действию
- **Подзаголовок блока действия** — подзаголовок (например: "приедем через 15-20 минут")
- **Изображение блока действия** — изображение для блока
- **Информация блока действия** — дополнительная информация

#### FAQ (по умолчанию)
- **Заголовок FAQ** — заголовок блока часто задаваемых вопросов
- **Ссылки FAQ** — список ссылок на вопросы
  - Текст ссылки
  - Ссылка
  - Стиль (белая кнопка / обычная кнопка)
  - Открывать в попапе (да/нет)
- **Элементы FAQ** — список вопросов и ответов
  - Вопрос
  - Ответ (с поддержкой HTML)

#### Архивы
- **Заголовок архива отзывов** — заголовок страницы со всеми отзывами
- **Описание архива отзывов** — описание страницы отзывов
- **Заголовок архива статей** — заголовок страницы со всеми статьями
- **Описание архива статей** — описание страницы статей

#### SEO настройки
- **Meta Keywords (по умолчанию)** — ключевые слова для SEO
- **Meta Description (по умолчанию)** — описание для поисковых систем
- **Open Graph изображение (по умолчанию)** — изображение для социальных сетей
- **Yandex Verification** — код верификации Яндекс.Вебмастер

#### Иконки сайта
- **Favicon 32x32** — иконка сайта 32x32 пикселя
- **Favicon 192x192** — иконка для Android 192x192 пикселя
- **Apple Touch Icon 180x180** — иконка для iOS 180x180 пикселей
- **MS Tile Image 270x270** — иконка для Windows 270x270 пикселей
- **URL логотипа для Schema.org** — логотип для структурированных данных

## Главная страница

**Путь:** Страницы → Главная (или страница, установленная как главная)

### Поля ACF для главной страницы:

#### Основные блоки
- **main_title** — главный заголовок страницы
  - Пример: "Профессиональное вскрытие замков в Москве и Подмосковье: автомобили, квартиры, сейфы"

#### Услуги на главной (main_services)
Повторяющееся поле с элементами:
- **title** — название услуги (например: "Вскрытие автомобилей")
- **image** — изображение услуги (опционально)
- **tags** — список тегов/ссылок (повторяющееся поле)
  - **text** — текст ссылки
  - **link** — URL ссылки
  - **half_width** — занимает половину ширины (да/нет)

#### Цены (prices)
Повторяющееся поле с элементами:
- **title** — название услуги (например: "Стоимость вскрытия автомобилей")
- **price** — цена (например: "от 1500 Р")
- **link** — ссылка на страницу услуги

#### Информационный текст
- **main_info_text** — текст с описанием компании (поддерживает HTML)

#### Преимущества
- **advantages_title** — заголовок блока преимуществ
- **advantages** — список преимуществ (повторяющееся поле)
  - **text** — текст преимущества (поддерживает HTML)

#### Заголовки секций
- **services_title** — заголовок блока услуг
- **reviews_title** — заголовок блока отзывов

#### FAQ
- **faq_title** — заголовок блока FAQ
- **faq_items** — список вопросов и ответов (повторяющееся поле)
  - **question** — вопрос
  - **answer** — ответ (поддерживает HTML)

## Услуги

### Создание услуги

**Путь:** Услуги → Добавить новую

#### Основные поля:
- **Заголовок** — название услуги (например: "Вскрытие автомобилей")
- **Контент** — основной текст услуги (опционально, если используется ACF поле `article_content`)
- **Краткое описание** — краткое описание для карточек услуг
- **Изображение записи** — главное изображение услуги

#### Поля ACF для услуги:

##### Информационные блоки (info_items)
Повторяющееся поле с элементами:
- **title** — заголовок (например: "24/7")
- **text** — описание (например: "Мастера дежурят круглосуточно")
- **icon** — иконка (изображение)

##### Описание услуги
- **service_description** — подробное описание услуги (поддерживает HTML)
- **service_price** — цена услуги (например: "от 1500 Р")

##### Контент статьи
- **article_content** — основной контент страницы услуги (поддерживает HTML)

##### Блок действия
- **action_title** — заголовок блока с призывом к действию
- **action_subtitle** — подзаголовок
- **action_image** — изображение для блока
- **action_info** — дополнительная информация

##### О компании/услуге
- **about_content** — контент блока "О нас" (поддерживает HTML)
- **about_image** — изображение для блока "О нас"

##### Преимущества
- **advantages_title** — заголовок блока преимуществ
- **advantages_subtitle** — подзаголовок (опционально)
- **advantages** — список преимуществ (повторяющееся поле)
  - **text** — текст преимущества (поддерживает HTML)

##### Заголовки секций
- **reviews_title** — заголовок блока отзывов
- **related_services_title** — заголовок блока связанных услуг
- **articles_title** — заголовок блока статей

##### FAQ
- **faq_title** — заголовок блока FAQ
- **faq_links** — ссылки на вопросы (повторяющееся поле)
  - **text** — текст ссылки
  - **link** — URL
  - **popup** — открывать в попапе (да/нет)
- **faq_items** — вопросы и ответы (повторяющееся поле)
  - **question** — вопрос
  - **answer** — ответ (поддерживает HTML)

### Страницы услуг (отдельные страницы)

**Путь:** Страницы → Все страницы

Для страниц услуг используется шаблон **"Страница услуги"**.

Примеры страниц:
- Вскрытие автомобилей
- Вскрытие двери квартиры
- Вскрытие сейфов
- Вскрытие гаражей

**Поля ACF** аналогичны полям для записей типа "Услуги".

## Отзывы

### Создание отзыва

**Путь:** Отзывы → Добавить новый

#### Основные поля:
- **Заголовок** — имя автора отзыва (например: "Елена")
- **Контент** — текст отзыва
- **Изображение записи** — фотография (опционально)

#### Поля ACF:
- **car_brand** — марка автомобиля (выбор из списка в настройках)
- **opening_time** — время вскрытия (выбор из списка в настройках)

### Публикация отзывов

Отзывы создаются со статусом **"На модерации"** (pending). Для публикации:
1. Перейдите в **Отзывы → Все отзывы**
2. Найдите отзыв
3. Измените статус на **"Опубликовано"**

### Форма отправки отзыва

Пользователи могут отправлять отзывы через форму на странице архива отзывов. Отзывы автоматически создаются со статусом "На модерации".

## Статьи

**Путь:** Статьи → Добавить новую

#### Основные поля:

- Безопасность жилища
- Все о сейфах

- Техпомощь

#### Поля ACF:
- **article_description** — краткое описание статьи (используется вместо стандартного excerpt)

## Страница контактов

**Путь:** Страницы → Контакты

### Поля ACF для страницы контактов:

#### Основные поля:
- **contacts_title** — заголовок страницы
- **contacts_description** — описание (например: "Мы работаем по Москве и Московской области.")

#### Контакты (contacts)
Повторяющееся поле с элементами:
- **title** — заголовок (например: "Телефоны в Москве:")
- **value** — значение (например: `<a href="tel:84955145350">+7(495) 514-53-50</a>`)
- **link** — ссылка на карту (опционально)

#### Карта
- **map_code** — код карты (например, Яндекс.Карты). Если не указан, используется карта по умолчанию.

## Меню навигации

**Путь:** Внешний вид → Меню

### Доступные области меню:

1. **Главное меню** — основное меню сайта
2. **Услуги в футере** — меню услуг в нижней части сайта
3. **Информация в футере** — информационное меню в футере

### Создание меню:

1. Создайте новое меню или выберите существующее
2. Добавьте нужные страницы/ссылки
3. Назначьте меню нужной области
4. Сохраните изменения

## Рекомендации по наполнению

### Изображения
- Используйте качественные изображения
- Оптимизируйте изображения перед загрузкой
- Рекомендуемый формат: JPG для фотографий, PNG для иконок
- Размер изображений: не более 2 МБ

### Тексты
- Пишите уникальные тексты для каждой страницы
- Используйте ключевые слова естественным образом
- Структурируйте текст заголовками (H2, H3)
- Добавляйте списки для лучшей читаемости

### SEO
- Заполняйте Meta Description для каждой страницы
- Используйте релевантные ключевые слова
- Создавайте уникальные заголовки (H1) для каждой страницы
- Добавляйте alt-тексты для всех изображений

### Отзывы
- Регулярно публикуйте новые отзывы
- Модерируйте отзывы перед публикацией
- Отвечайте на отзывы (если есть такая функция)

### Статьи
- Публикуйте статьи регулярно (1-2 раза в неделю)
- Используйте категории для организации статей
- Добавляйте релевантные изображения к статьям
- Связывайте статьи с услугами

## Часто задаваемые вопросы

### Как изменить телефон на сайте?
Перейдите в **Настройки сайта → Основной телефон** и укажите новый номер.

### Как добавить новую услугу?
1. Перейдите в **Услуги → Добавить новую**
2. Заполните основные поля
3. Заполните поля ACF
4. Опубликуйте услугу

### Как изменить порядок услуг?
В списке услуг используйте перетаскивание (drag & drop) или установите порядок через поле "Порядок" (menu_order).

### Как добавить отзыв?
1. Перейдите в **Отзывы → Добавить новый**
2. Заполните поля
3. Опубликуйте отзыв

### Как изменить контакты?
Перейдите в **Страницы → Контакты** и отредактируйте поля ACF.

### Где настраивается меню?
Перейдите в **Внешний вид → Меню** и создайте/отредактируйте нужное меню.
